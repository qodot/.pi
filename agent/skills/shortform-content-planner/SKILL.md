# 숏폼 마케팅 컨텐츠 기획서 생성기

이커머스 제품에 대한 인플루언서 마케팅용 숏폼 컨텐츠 기획서를 체계적으로 생성하는 스킬입니다.

## 의존 도구 및 스킬

이 스킬은 다음 도구/스킬들과 함께 사용합니다:

**필수 확장:**
- `ask`: 사용자에게 객관식 질문 (단일)
- `ask_multi`: 사용자에게 여러 객관식 질문 (순차적)

**권장 스킬:**
- `brave-search`: 브랜드 조사, 트렌드 분석, 경쟁사 분석
- `youtube-transcript`: 인기 숏폼 컨텐츠 분석

---

## 프로세스

### 1단계: 기본 정보 수집

사용자에게 다음 **3가지만** 먼저 수집합니다.

> 💡 **ask 확장 사용**: 브랜드명은 `ask` 도구로 수집하고, PDP URL은 직접 입력을 요청합니다.

```
📦 기본 정보 (필수)
1. 브랜드명:
2. PDP 1: (메인 제품)
3. PDP 2: (추가 제품 또는 같은 제품의 다른 채널)
```

### 2단계: 브랜드 및 제품 자동 조사

**brave-search로 브랜드 조사**:
```bash
/skill:brave-search "[브랜드명]" --content
/skill:brave-search "[브랜드명] 브랜드 스토리" --content
```
- 브랜드 카테고리, 포지셔닝, 타겟층 추론

**URL에서 제품 정보 추출** (WebFetch 또는 brave-search content.js):
```bash
# brave-search 스킬의 content.js 활용
{brave-search-dir}/content.js [PDP1]
{brave-search-dir}/content.js [PDP2]
```
- 제품명, 가격, 주요 성분/특징, USP 파악
- 제품 카테고리 추론

### 3단계: 추론 결과 확인

조사 결과를 바탕으로 **추론한 내용을 사용자에게 확인**합니다.

> 💡 **ask 확장 사용**: 분석 결과를 표로 보여준 후, `ask` 도구로 확인을 받습니다.
> - 질문: "분석 결과가 맞나요?"
> - 옵션: ["맞습니다", "수정이 필요합니다"]

```
🔍 분석 결과 확인

제가 파악한 내용이 맞는지 확인해주세요:

| 항목 | 추론 결과 |
|------|----------|
| 브랜드 카테고리 | {화장품/F&B/패션/테크 등} |
| 제품 유형 | {세럼/크림/음료/의류 등} |
| 예상 타겟층 | {20-30대 여성, 스킨케어에 관심 등} |
| 제품 USP | {주요 차별점} |

👉 수정할 내용이 있으면 알려주세요!
```

### 4단계: 캠페인 정보 수집

추론이 확인되면 나머지 정보를 수집합니다.

> 💡 **ask_multi 확장 사용**: 다음 질문들을 `ask_multi` 도구로 순차적으로 수집합니다.
> ```
> questions: [
>   { key: "country", question: "대상 국가는?", options: ["한국", "미국", "일본", "동남아"] },
>   { key: "goal", question: "캠페인 목표는?", options: ["인지도 향상", "전환 유도", "브랜드 이미지 구축", "신규 고객 확보"] },
>   { key: "platform", question: "주요 플랫폼은?", options: ["TikTok", "Instagram Reels", "YouTube Shorts"] },
>   { key: "product_provision", question: "제품 제공 여부는?", options: ["제공", "미제공"] }
> ]
> ```
> - 타겟 오디언스와 리워드 조건은 직접 입력을 요청합니다.

```
📋 캠페인 정보

1. 대상 국가: (한국 / 미국 / 일본 / 동남아 / 기타)
2. 캠페인 목표: (인지도 향상 / 전환 유도 / 브랜드 이미지 구축 / 신규 고객 확보)
3. 타겟 오디언스: (연령대, 성별) - 추론과 다르면 수정
4. 주요 플랫폼: (TikTok / Instagram Reels / YouTube Shorts / 복수 선택)
5. 제품 제공 여부: (제공 / 미제공)
6. 리워드 조건: (금액, 지급 방식)
```

## 국가별 플랫폼 특성

| 국가 | 주요 플랫폼 | 특성 |
|------|-------------|------|
| **한국** | Instagram, YouTube | 네이버 블로그 연계, 뷰티/패션 강세 |
| **미국** | TikTok, Instagram | 다양성 중시, UGC 선호 |
| **일본** | YouTube, TikTok | 정제된 콘텐츠 선호, 상세 설명 중요 |
| **동남아** | TikTok, Shopee Live | 라이브커머스 강세, 가성비 강조 |

## 리워드별 컨텐츠 엄격성 기준

| 리워드 수준 | 컨텐츠 엄격성 | 가이드라인 특성 |
|-------------|---------------|-----------------|
| **높음** (30만원+) | 엄격함 | 필수 멘트, 정해진 구성, 리뷰 필수 |
| **중간** (10~30만원) | 보통 | 핵심 메시지 전달, 일부 자유도 |
| **낮음** (10만원 미만) | 자유로움 | 크리에이터 스타일 존중, 최소 가이드 |
| **제품만 제공** | 매우 자유로움 | 솔직한 후기, 창의적 표현 권장 |

## 5단계: 트렌드 분석 (최근 3개월)

**brave-search로 트렌드 조사**:

```bash
# 카테고리별 숏폼 트렌드
/skill:brave-search "[국가] [카테고리] 숏폼 트렌드 2026" --freshness pm
/skill:brave-search "[국가] [카테고리] TikTok viral" --freshness pm
/skill:brave-search "[국가] [플랫폼] [카테고리] 인기 컨텐츠" --freshness pm

# 경쟁사/참고 사례
/skill:brave-search "[경쟁브랜드] 인플루언서 마케팅" --content
/skill:brave-search "[카테고리] 바이럴 영상 사례" --freshness pm
```

**분석 항목**:
- 현재 인기 있는 컨텐츠 포맷
- 트렌딩 해시태그
- 바이럴 훅/밈
- 경쟁 브랜드 캠페인 사례
- 플랫폼별 알고리즘 변화

**충분한 분석 기준**:
- 최소 5개 이상의 관련 트렌드 파악
- 3개 이상의 경쟁/참고 사례 수집
- 플랫폼별 최신 알고리즘 특성 확인

## 6단계: 기존 기획서 참고

기획서 작성 전, **유사한 카테고리/국가의 기존 기획서**를 참고하여 톤앤매너, 구성, 기획 품질을 맞춥니다.

### 참고 기획서 목록

| 파일명 | 카테고리 | 국가 | 특징 |
|--------|----------|------|------|
| `[메디필_미국] 바이럴 영상 기획안.pdf` | 화장품 (스킨케어) | 미국 | 미국 시장 타겟 기획 |
| `[웰라쥬] 바이럴 영상 기획안_4차.pdf` | 화장품 (스킨케어) | - | 4차 수정본, 정제된 기획 |
| `[이니스프리] 바이럴 영상 기획안.pdf` | 화장품 (스킨케어) | - | 대형 브랜드 기획 사례 |
| `[비비드로우] 바이럴 기획안.pdf` | 화장품 (색조) | - | 색조 화장품 기획 |
| `[더후] 선라인 기획안.pdf` | 화장품 (럭셔리) | - | 럭셔리 브랜드 톤앤매너 |
| `[아비브] 바이럴 기획안.pdf` | 화장품 (스킨케어) | - | 스킨케어 기획 |
| `[블랑두부] 콘텐츠 기획안.pdf` | F&B (식품) | - | 식품 카테고리 기획 |
| `[에스쁘아] 1차 - Be velvet Cover Cushion.pdf` | 화장품 (색조) | - | 쿠션 제품 기획 |
| `[에스쁘아] 2차 - 뉴 비벨벳 커버쿠션.pdf` | 화장품 (색조) | - | 2차 수정본 |
| `[불스원] 2차 기획안.pdf` | 자동차용품 | - | 비화장품 카테고리 참고 |

### 참고 방법

1. **카테고리 매칭**: 현재 작업 중인 브랜드와 유사한 카테고리의 기획서 1-2개 선택
2. **PDF 읽기**: `read` 도구로 해당 PDF 파일 확인
   ```
   examples/[브랜드명] 기획안.pdf
   ```
3. **참고 포인트**:
   - Campaign Summary 표 구성 방식
   - 콘텐츠 상세 기획의 톤앤매너
   - 핵심 포인트 작성 스타일
   - 영상 흐름 구성 방식
   - 해시태그 전략

> ⚠️ **주의**: 기존 기획서는 **형식과 품질 수준 참고용**입니다. 컨텐츠 아이디어나 기획 내용을 그대로 복사하지 마세요. 반드시 현재 브랜드/제품에 맞는 새로운 기획을 작성해야 합니다.

## 7단계: 컨텐츠 전략 수립

수집된 정보와 트렌드 분석을 바탕으로 결정:

- **훅(Hook)**: 첫 3초 내 시청자를 사로잡을 요소 (트렌드 반영)
- **핵심 메시지**: 전달할 1가지 명확한 메시지
- **CTA**: 시청 후 유도할 행동
- **컨텐츠 포맷**: 튜토리얼 / 리뷰 / 챌린지 / 일상 브이로그 / 비포애프터 (트렌드 반영)

### 컨텐츠 훅(Hook) 유형

**질문형 훅**
- "아직도 {문제}로 고민하세요?"
- "{제품}없이 {활동}하고 계신가요?"

**충격형 훅**
- "{숫자}명이 이미 바꿨습니다"
- "이거 모르면 {금액} 손해봅니다"

**비교형 훅**
- "vs {경쟁제품}, 직접 비교해봤습니다"
- "Before & After 보세요"

**스토리형 훅**
- "처음엔 저도 믿지 않았어요"
- "친구가 추천해서 써봤는데..."

**트렌드형 훅**
- "요즘 {플랫폼}에서 난리난 {제품}"
- "{인플루언서}도 쓰는 {제품}"

### 컨텐츠 포맷 아이디어

**제품 중심 포맷**
1. 언박싱: 첫 개봉 리액션
2. ASMR: 제품 사용 사운드
3. Before/After: 사용 전후 비교
4. 하루 루틴: 일상 속 제품 활용
5. 꿀팁 공유: 숨겨진 사용법

**스토리텔링 포맷**
1. POV: 특정 상황 시점
2. Storytime: 제품 발견 스토리
3. 챌린지: 제품 관련 챌린지 참여/생성

**정보성 포맷**
1. Top 3/5: 제품 특징 리스트
2. 비교 리뷰: 타 제품과 비교
3. Q&A: 자주 묻는 질문 답변
4. 사용 튜토리얼: 단계별 사용법

## 8단계: 기획서 작성

[templates/content-plan-template.md](templates/content-plan-template.md) 템플릿을 기반으로 기획서를 작성합니다.

> ⚠️ **중요: 예시 기획 복사 금지**
> 
> 템플릿의 예시는 **형식과 톤앤매너 참고용**입니다. 컨텐츠명, 기획 내용, 핵심 포인트, 영상 흐름 등을 그대로 복사하거나 살짝만 변형하여 사용하는 것은 **절대 금지**합니다. 반드시 트렌드 분석 결과와 해당 브랜드/제품 특성에 맞는 **새로운 기획**을 작성해야 합니다.

## 품질 보증 체크리스트

기획서 작성 완료 후 다음 항목을 확인합니다:

- [ ] 브랜드 검색과 PDP 분석이 완료되었는가?
- [ ] 추론한 카테고리/타겟층이 사용자에게 확인되었는가?
- [ ] 최근 3개월 트렌드 분석이 충분히 수행되었는가? (5개+ 트렌드, 3개+ 사례)
- [ ] **유사 카테고리의 기존 기획서를 참고했는가?**
- [ ] Campaign Summary 표에 3개 이상의 컨텐츠가 정리되었는가?
- [ ] 각 콘텐츠 상세 기획에 크리에이터 유형이 명시되었는가?
- [ ] 각 콘텐츠에 핵심 포인트 3개와 영상 흐름이 포함되었는가?
- [ ] 영상 흐름에 실제 사용 가능한 예시 멘트/썸네일 텍스트가 있는가?
- [ ] 플랫폼별 특성이 반영되었는가?
- [ ] 국가별 문화적 특성이 반영되었는가?
- [ ] 리워드 수준에 맞는 가이드라인 엄격성이 설정되었는가?

## 오류 처리

| 오류 상황 | 대응 |
|----------|------|
| **PDP 접근 불가** | 사용자에게 제품 정보 직접 입력 요청 (제품명, 주요 성분/특징, USP) |
| **브랜드 검색 결과 부족** | 사용자에게 브랜드 카테고리, 포지셔닝 직접 입력 요청 |
| **트렌드 검색 결과 부족** | 검색 키워드 변경 후 재시도, 관련 카테고리로 확장 |
| **필수 정보 누락** | 누락된 정보 명시하고 추가 수집 |
| **플랫폼 미지원 국가** | 해당 플랫폼 제외 후 대안 제시 |
